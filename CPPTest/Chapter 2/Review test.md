### 4．什么语句可以用来打印短语“Hello，world”，然后开始新的一行？

```
### 5．什么语句可以用来创建名为cheeses的整数变量？

```
### 6．什么语句可以用来将值32赋给变量cheeses？

### 7．什么语句可以用来将从键盘输入的值读入变量cheeses中？
```C++

```
### 8．什么语句可以用来打印“We have X varieties of cheese,”，其中X为变量cheeses的当前值。
```C++

```
