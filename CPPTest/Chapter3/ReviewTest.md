### 6．如何使用C++来找出编码88表示的字符？指出至少两种方法。
```C++
方法一:
#include <iostream>
int main() {
	char ct = 88;
	std::cout << "答案是" << ct << std::endl;
	return 0;
}
方法二:
#include <iostream>
int main() {
	int num = 88;
	char ct = static_cast<char>(num);
	std::cout << "答案是" << ct << std::endl;
	return 0;
}
```
